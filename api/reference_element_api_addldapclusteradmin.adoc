= 新增 LDAP 叢集管理員
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `AddLdapClusterAdmin`新增的 LDAP 叢集管理員使用者。  LDAP 叢集管理員可以使用 API 和管理工具來管理叢集。  LDAP 叢集管理員帳戶與標準租用戶帳戶完全獨立且無關。



== 參數

您也可以使用此方法新增已在 Active Directory® 中定義的 LDAP 群組。授予群組的存取等級會傳遞給 LDAP 群組中的各個使用者。

== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "AddLdapClusterAdmin",
  "params": {"username":"cn=mike jones,ou=ptusers,dc=prodtest,dc=solidfire,dc=net",
    "access": ["administrator", "read"
    ]
  },
"id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
 "id": 1,
 "result": {}
}
----
