= 取得網路介面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNetworkInterface`取得節點上網路介面資訊的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
介面
 a| 
取得每個節點資訊的介面名稱。可能的值：

* Bond1G
* Bond10G

 a| 
細繩
 a| 
沒有任何
 a| 
不



 a| 
力量
 a| 
將此參數設為 true 可在叢集中的所有節點上運行。
 a| 
布林值
 a| 
錯誤的
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
節點
 a| 
描述儲存叢集中每個儲存節點介面的物件數組。數組中的每個物件都包含以下項目：

* nodeID: (整數) 介面資訊所適用的儲存叢集中儲存節點的 ID。
* 結果： （xref:reference_element_api_networkinterface.adoc[網路介面] ）此儲存節點的介面配置資訊。

 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
	"method": "GetNetworkInterface",
	"params": {
		"interface": "Bond1G",
		"force": true
		},
	"id": 1
}
----
