= 刪除儲存容器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteStorageContainers`一次從系統中刪除多達 2000 個虛擬磁碟區 (VVol) 儲存容器的方法。刪除的儲存容器不得包含任何虛擬磁碟區 (VVol)。



== 參數

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
儲存容器 ID
 a| 
要刪除的儲存容器 ID 清單。清單中最多可以指定 2000 個 ID。
 a| 
UUID 數組
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "DeleteStorageContainers",
    "params": {
        "storageContainerIDs" : ["a9ec1138-e386-4a44-90d7-b9acbbc05176"]
    },
    "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 1,
  "result": {}
}
----
