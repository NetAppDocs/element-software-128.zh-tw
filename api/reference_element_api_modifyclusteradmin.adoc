= 修改叢集管理員
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ModifyClusterAdmin`變更叢集管理員、LDAP 叢集管理員或第三方身分提供者 (IdP) 叢集管理員的設定的方法。您無法變更管理員群集管理員帳戶的存取權限。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
使用權
 a| 
控制此集群管理員可以使用的方法。
 a| 
字串數組
 a| 
沒有任何
 a| 
不



 a| 
屬性
 a| 
JSON 物件格式的名稱-值對清單。
 a| 
JSON 物件
 a| 
沒有任何
 a| 
不



 a| 
叢集管理員ID
 a| 
要修改的叢集管理員、LDAP 叢集管理員或 IdP 叢集管理員的 ClusterAdminID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
密碼
 a| 
用於驗證此群集管理員身分的密碼。此參數不適用於 LDAP 或 IdP 叢集管理員。
 a| 
細繩
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ModifyClusterAdmin",
   "params": {
     "clusterAdminID"  : 2,
     "password"   : "7925Brc429a"
   },
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
 "id" : 1
 "result" : { }
}
----
