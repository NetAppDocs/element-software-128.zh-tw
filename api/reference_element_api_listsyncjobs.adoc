= 清單同步作業
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListSyncJobs`取得有關 Element 儲存叢集上執行的同步作業的資訊的方法。此方法傳回有關切片、複製、區塊和遠端同步作業的資訊。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
同步作業
 a| 
描述系統中目前正在運行的同步進程的物件清單。
 a| 
xref:reference_element_api_syncjob.adoc[同步作業]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListSyncJobs",
   "params": { },
   "id" : 1
}
----
