= 取得NTP訊息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNtpInfo`取得目前網路時間協定（NTP）配置資訊的方法。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
伺服器
 a| 
NTP伺服器清單。
 a| 
字串數組



 a| 
廣播客戶端
 a| 
指示叢集中的節點是否正在監聽廣播 NTP 訊息。可能的值：

* 真的
* 錯誤的

 a| 
布林值

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "GetNtpInfo",
  "params": {},
  "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {
    "broadcastclient" : false,
    "servers" : [ "us.pool.ntp.org" ]
   }
}
----
