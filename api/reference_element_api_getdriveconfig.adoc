= 取得驅動配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDriveConfig`取得預期切片和區塊驅動器數量的驅動器信息，以及當前連接到節點的切片和區塊驅動器的數量的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
驅動配置
 a| 
連接到節點的驅動器資訊。
 a| 
xref:reference_element_api_drive.adoc[駕駛]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetDriveConfig",
   "params": {},
   "id" : 1
}
----
