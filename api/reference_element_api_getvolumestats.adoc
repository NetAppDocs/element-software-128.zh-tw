= 獲取成交量統計信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetVolumeStats`取得單一體積的高級活性測量值的方法。此數值自卷創建以來不斷累積。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
卷 ID
 a| 
指定要收集統計資料的磁碟區。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
銷量統計
 a| 
成交量活動資訊。
 a| 
xref:reference_element_api_volumestats.adoc[銷量統計]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetVolumeStats",
   "params": {
      "volumeID": 32
   },
   "id": 1
}
----
