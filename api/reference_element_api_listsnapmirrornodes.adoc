= 列出快照鏡像節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `ListSnapMirrorNodes`取得遠端ONTAP叢集中節點清單的方法。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
如果提供了 snapMirrorEndpointID，系統將列出具有該 snapMirrorEndpointID 的端點的節點。如果未提供，系統將列出所有已知SnapMirror端點的節點。
 a| 
整數
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
snapMirrorNodes
 a| 
ONTAP叢集上的節點清單。
 a| 
xref:reference_element_api_snapmirrornode.adoc[snapMirrorNode]大批

|===
