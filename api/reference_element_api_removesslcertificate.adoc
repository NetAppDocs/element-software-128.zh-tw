= 移除SSL憑證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RemoveSSLCertificate`刪除叢集中儲存節點的使用者 SSL 憑證和私鑰的方法。刪除憑證和私鑰後，儲存節點將配置為使用預設憑證和私鑰。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
    "method" : "RemoveSSLCertificate",
    "params" : {},
    "id" : 3
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
    "id" : 3,
    "result" : {}
}
----
