= 列出活動節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListActiveNodes`傳回叢集中目前活動節點清單的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
節點
 a| 
集群中活躍節點的列表。
 a| 
xref:reference_element_api_node.adoc[節點]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListActiveNodes",
   "params": {},
   "id" : 1
}
----
