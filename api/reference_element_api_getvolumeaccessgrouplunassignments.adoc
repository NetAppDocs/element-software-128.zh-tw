= 取得卷宗訪問群組 LUN 分配
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetVolumeAccessGroupLunAssignments`檢索指定磁碟區存取群組的 LUN 對應詳細資訊的方法。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
卷訪問群組ID
 a| 
用於傳回資訊的唯一磁碟區存取群組 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
卷訪問群組 LUN 分配
 a| 
所有實體光纖通道連接埠的列表，或單一節點的連接埠列表。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetVolumeAccessGroupLunAssignments",
   "params": {
      "volumeAccessGroupID": 5
    },
  "id" : 1
   }
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {
    "volumeAccessGroupLunAssignments" : {
       "volumeAccessGroupID" : 5,
       "lunAssignments" : [
          {"volumeID" : 5, "lun" : 0},
          {"volumeID" : 6, "lun" : 1},
          {"volumeID" : 7, "lun" : 2},
          {"volumeID" : 8, "lun" : 3}
       ],
       "deletedLunAssignments" : [
           {"volumeID" : 44, "lun" : 44}
       ]
    }
  }
}
----
