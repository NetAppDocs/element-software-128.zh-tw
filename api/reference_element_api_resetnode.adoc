= 重置節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ResetNode`將節點重設為出廠設定的方法。呼叫此方法時，節點中的所有資料、軟體包（軟體升級等）、配置和日誌檔案都將被刪除。但是，在此操作過程中，節點的網路設定將被保留。參與叢集的節點無法重設為出廠設定。



== 參數

ResetNode API 只能用於處於「可用」狀態的節點。它不能用於叢集中處於「活動」狀態或「待定」狀態的節點。

警告：

此方法會清除節點上的所有客戶資料。

== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ResetNode",
   "params": {
      "build" : "file:///sf/rtfi/image/filesystem.squashfs",
      "force" : true
              },
   "id" : 1
}
----
