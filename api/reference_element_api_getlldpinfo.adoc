= 取得Lldp訊息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetLldpInfo`取得儲存叢集中每個節點或單一儲存節點的鏈結層發現協定 (LLDP) 配置的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
lldpInfo
 a| 
儲存叢集中每個節點的機箱、介面和相鄰 LLDP 設定資訊。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetLldpInfo",
   "id" : 1
}
----
