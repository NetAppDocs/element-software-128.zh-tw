= 列出已刪除卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListDeletedVolumes`取得已標記為刪除並從系統中清除的磁碟區清單的方法。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
包含虛擬卷
 a| 
預設情況下，回應中包含虛擬磁碟區。若要排除虛擬卷，請設定為 false。
 a| 
布林值
 a| 
真的
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
卷
 a| 
已刪除磁碟區清單。
 a| 
xref:reference_element_api_volume.adoc[體積]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListDeletedVolumes",
   "params": {},
   "id" : 1
}
----
