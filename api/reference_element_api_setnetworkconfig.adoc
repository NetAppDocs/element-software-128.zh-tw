= 設定網路配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `SetNetworkConfig`設定節點網路配置的方法。若要顯示節點的目前網路設置，請執行下列命令 `GetNetworkConfig`API 方法。



== 範圍

警告：

更改節點上的綁定模式可能會導致網路連線暫時中斷。

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
網路
 a| 
包含要修改的節點網路設定的物件。您只需將要變更的欄位作為屬性新增至此方法的參數即可。
 a| 
xref:reference_element_api_network_all_interfaces.adoc[網路（所有介面）]
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
網路
 a| 
節點的最新網路配置。
 a| 
xref:reference_element_api_network_all_interfaces.adoc[網路（所有介面）]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
	"method": "SetNetworkConfig",
		"params": {
			"network": {
				"Bond10G": {
					"bond-mode": "ALB"
				},
				"Bond1G": {
					"netmask": "255.255.224.0"
				},
				"eth0": {
					"method": "bond"
				},
				"lo": {
					"method": "loopback"
				}
			}
		}
}
----


== 回應範例

此方法的回應與 GetNetworkConfig 方法的回應相同。此方法顯示每個物件的所有成員，並包含任何已更改成員的新值。
