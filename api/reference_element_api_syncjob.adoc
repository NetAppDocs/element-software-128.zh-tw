= 同步作業
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
syncJob 物件包含在叢集上執行的複製、遠端複製或切片同步作業的資訊。

您可以使用以下方式檢索同步資訊： `ListSyncJobs` API 方法。



== 對象成員

該物件包含以下成員：

|===
| Name | 描述 | 類型 


 a| 
每秒區塊數
 a| 
每秒從來源集群傳輸到目標集群的資料塊數量。僅當類型成員設定為遠端時才存在。
 a| 
整數



 a| 
分支類型
 a| 
僅用於遠端複製同步作業。可能的值：

* 快照
* 體積

 a| 
細繩



 a| 
每秒位元組數
 a| 
克隆進程每秒處理的位元組數。僅當類型成員設定為克隆或切片時才存在。
 a| 
漂浮



 a| 
複製ID
 a| 
正在進行的克隆操作的標識符。僅當類型成員設定為克隆時才存在。
 a| 
整數



 a| 
目前位元組數
 a| 
克隆檔案在來源磁碟區中已處理的位元組數。僅當類型成員設定為克隆或切片時才存在。
 a| 
整數



 a| 
目標服務ID
 a| 
託管該卷主副本的服務識別碼。僅當類型成員設定為遠端時才存在。
 a| 
整數



 a| 
目標磁碟區 ID
 a| 
目標卷 ID。僅當類型成員設定為克隆或遠端時才存在。
 a| 
整數



 a| 
經過時間
 a| 
同步任務開始後經過的時間（以秒為單位）。
 a| 
根據同步操作的類型，可以是浮點數或整數。



 a| 
groupCloneID
 a| 
正在進行的群組克隆操作的 ID。
 a| 
整數



 a| 
節點ID
 a| 
指定克隆操作發生的節點。僅當類型成員設定為克隆時才存在。
 a| 
整數



 a| 
完成百分比
 a| 
同步作業完成百分比。
 a| 
根據同步操作的類型，可以是浮點數或整數。



 a| 
剩餘時間
 a| 
完成操作的預計時間（以秒為單位）。
 a| 
漂浮



 a| 
切片ID
 a| 
正在同步的切片磁碟機的 ID。
 a| 
整數



 a| 
階段
 a| 
僅當類型成員設定為遠端或克隆時才存在。可能的值：

* 元資料：複製過程正在確定需要將哪些資料傳輸到遠端叢集。複製過程的此階段不報告狀態。
* 資料：複製過程正在將大部分資料傳輸到遠端叢集。
* 整體：表示切片同步作業的向後相容性。

 a| 
細繩



 a| 
snapshotID
 a| 
建立克隆所依據的快照的 ID。僅當類型成員設定為克隆時才存在。
 a| 
整數



 a| 
srcServiceID
 a| 
來源服務 ID。
 a| 
整數



 a| 
srcVolumeID
 a| 
來源磁碟區 ID。
 a| 
整數



 a| 
總位元組數
 a| 
克隆檔案的總位元組數。僅當類型成員設定為克隆或切片時才存在。
 a| 
整數



 a| 
類型
 a| 
同步操作的類型。可能的值：

* 複製
* 片
* 堵塞
* 偏僻的

 a| 
細繩

|===
