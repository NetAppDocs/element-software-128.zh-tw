= 禁用SSH
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DisableSsh`禁用單一儲存節點的 SSH 服務的方法。此方法不會影響叢集範圍內的 SSH 服務逾時時長。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
已啟用
 a| 
此節點的 SSH 服務狀態。
 a| 
布林值

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "DisableSsh",
  "params": {
      },
  "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": false}
}
----
