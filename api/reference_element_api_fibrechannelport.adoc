= 光纖通道連接埠
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
fiberChannelPort 物件包含有關節點上的各個連接埠或叢集中整個節點的資訊。您可以使用以下方法檢索此資訊： `ListNodeFibreChannelPortInfo`方法。

該物件包含以下成員：

|===
| Name | 描述 | 類型 


 a|

 a| 
光纖通道連接埠上安裝的韌體版本。
 a| 
整數



 a| 
hbaPort
 a| 
單一主機匯流排適配器（HBA）連接埠的 ID。
 a| 
整數



 a| 
模型
 a| 
連接埠上的HBA型號。
 a| 
細繩



 a| 
nPortID
 a| 
唯一的連接埠節點ID。
 a| 
細繩



 a| 
PCI插槽
 a| 
光纖通道節點機箱中容納 PCI 卡的插槽。
 a|

 a| 
序號
 a| 
光纖通道連接埠上的序號。
 a| 
細繩



 a|
