= 列出叢集介面首選項
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
這 `ListClusterInterfacePreference`此方法使與運行 Element 軟體的儲存叢集整合的系統能夠列出儲存在系統中的現有叢集介面首選項。此方法僅供內部使用。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
偏好
 a| 
目前儲存在儲存叢集上的叢集介面物件列表，每個物件包含首選項的名稱和值。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
	"method": "ListClusterInterfacePreferences",
    "params": {
    },
    "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
	"id": 1,
	"result": {
		"preferences": [
			{
				"name": "prefname",
				"value": "testvalue"
			}
		]
	}
}
----
