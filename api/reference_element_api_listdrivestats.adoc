= ListDriveStats
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListDriveStats`列出集群中多個驅動器的高級活動測量值的方法。預設情況下，此方法會傳回叢集中所有磁碟機的統計訊息，這些測量值是從將磁碟機新增至叢集開始累積的。此方法傳回的一些值是區塊磁碟機特有的，而有些值是元資料磁碟機特有的。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
驅動
 a| 
若要傳回磁碟機統計資料的磁碟機 ID (driveID) 清單。如果省略此參數，則會傳回所有磁碟機的測量結果。
 a| 
整數數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法具有以下傳回值：

|===


| Name | 描述 | 類型 


 a| 
驅動統計
 a| 
每個磁碟機的活動資訊清單。
 a| 
xref:reference_element_api_drivestats.adoc[驅動統計]大批



 a| 
錯誤
 a| 
此清單包含磁碟機 ID 和相關的錯誤訊息。它始終存在，如果沒有錯誤，則為空。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 1,
  "method": "ListDriveStats",
  "params": {
    "drives":[22,23]
  }
}
----
