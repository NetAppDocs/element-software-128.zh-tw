= ListSnapMirror關係
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `ListSnapMirrorRelationships`列出 Element 儲存叢集上一個或多個SnapMirror關係的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
僅列出與指定端點 ID 關聯的關係。如果沒有提供端點 ID，系統將列出所有已知SnapMirror端點的關係。
 a| 
整數
 a| 
沒有任何
 a| 
不



 a| 
目的地捲
 a| 
列出與指定目標磁碟區關聯的關係。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeInfo]
 a| 
沒有任何
 a| 
不



 a| 
來源磁碟區
 a| 
列出與指定來源磁碟區關聯的關係。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeInfo]
 a| 
沒有任何
 a| 
不



 a| 
虛擬伺服器
 a| 
列出指定虛擬伺服器上的關係。
 a| 
細繩
 a| 
沒有任何
 a| 
不



 a| 
關係ID
 a| 
列出與指定關係 ID 相關的關係。
 a| 
細繩
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
snapMirror關係
 a| 
包含有關SnapMirror關係資訊的物件清單。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror關係]大批

|===
