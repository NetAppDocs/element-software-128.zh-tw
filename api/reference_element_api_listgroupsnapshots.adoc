= 列出群組快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListGroupSnapshots`傳回有關所有已建立群組快照的資訊的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
群組快照ID
 a| 
取得單一群組快照 ID 的資訊。
 a| 
整數
 a| 
沒有任何
 a| 
不



 a| 
卷
 a| 
要查詢的唯一磁碟區 ID 陣列。如果您不指定此參數，則會包含叢集上的所有群組快照。
 a| 
volumeID 數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
群組快照
 a| 
包含每個群組快照資訊的物件清單。
 a| 
xref:reference_element_api_groupsnapshot.adoc[群組快照]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListGroupSnapshots",
  "params": {
  "volumes": [
     31,
     49
    ]
  },
  "id": 1
}
----
