= 快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
快照物件包含有關為磁碟區建立的快照的資訊。你可以使用 `ListSnapshots`用於檢索磁碟區或所有磁碟區的快照資訊清單的 API 方法。該物件包含有關活動快照以及為磁碟區建立的每個快照的資訊。



== 對象成員

該物件包含以下成員：

|===
| Name | 描述 | 類型 


 a| 
屬性
 a| 
JSON 物件格式的名稱-值對清單。
 a| 
JSON 物件



 a| 
校驗和
 a| 
儲存快照中資料的簡短字串表示形式。之後可以使用此校驗和來比較其他快照，以偵測資料中的錯誤。
 a| 
細繩



 a| 
創建時間
 a| 
快照建立時間（UTC+0格式）。
 a| 
ISO 8601 日期字串



 a| 
啟用遠端複製
 a| 
確定是否已為遠端複製啟用快照。
 a| 
布林值



 a| 
到期原因
 a| 
指示快照過期時間的設定方式。可能的值：

* API：過期時間可透過 API 設定。
* 無：未設定過期時間。
* 測試：設定有效期限進行測試。
* 先進先出：到期依先進先出原則進行。

 a| 
細繩



 a| 
過期時間
 a| 
此快照的過期時間以及從叢集中清除的時間。
 a| 
ISO 8601 日期字串



 a| 
組ID
 a| 
如果快照是某個群組快照的成員，則需要提供該群組 ID。
 a| 
整數



 a| 
群組快照 UUID
 a| 
包含有關群組中每個快照的資訊。每個成員都將有一個 UUID 參數，用於設定快照的 UUID。
 a| 
細繩



 a| 
實例建立時間
 a| 
在本地叢集上建立快照的時間。
 a| 
ISO 8601 日期字串



 a| 
實例快照 UUID
 a| 
本機叢集上快照的通用唯一 ID。此ID不會複製到其他叢集。
 a| 
細繩



 a| 
姓名
 a| 
分配給快照的唯一名稱。如果沒有指定名稱，則名稱為快照建立時的 UTC+0 格式時間戳記。
 a| 
細繩



 a| 
遠端狀態
 a| 
包含從來源叢集看到的、目標叢集上每個遠端快照的通用識別碼和複製狀態的陣列。
 a| 
xref:reference_element_api_remoteclustersnapshotstatus.adoc[遠端集群快照狀態]大批



 a| 
snapMirrorLabel
 a| 
SnapMirror軟體用於指定SnapMirror端點上的快照保留策略的標籤。如果未設置，則此值為空。
 a| 
細繩



 a| 
snapshotID
 a| 
現有快照的唯一 ID。
 a| 
細繩



 a| 
快照 UUID
 a| 
現有快照的通用唯一識別碼。當快照在群集間複製時，此 ID 也隨之複製，並用於在群集間識別快照。
 a| 
細繩



 a| 
地位
 a| 
快照的目前狀態。可能的值：

* 未知：取得快照狀態時發生錯誤。
* 準備中：此快照正在準備使用，目前尚不可寫入。
* 遠端同步：此快照正在從遠端叢集複製。
* 完成：此快照已完成準備或複製，現在可以使用了。
* 目前狀態：此快照為目前活動分支。
* 克隆：此快照參與了 CopyVolume 操作。

 a| 
細繩



 a| 
總大小
 a| 
快照的總大小（以位元組為單位）。
 a| 
整數



 a| 
虛擬磁碟區 ID
 a| 
與此快照關聯的虛擬磁碟區的 ID。
 a| 
唯一識別符



 a| 
卷 ID
 a| 
建立快照所依據的磁碟區的 ID。
 a| 
整數



 a| 
卷名
 a| 
建立快照時磁碟區的名稱。
 a| 
細繩
