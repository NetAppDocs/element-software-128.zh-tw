= 取得硬體資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetHardwareInfo`取得單一節點的即時硬體資訊和狀態的方法。硬體資訊通常包括製造商、供應商、版本、驅動器和其他相關標識資訊。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
力量
 a| 
將此“force”參數設為 true，即可在叢集中的所有節點上運行。
 a| 
布林值
 a| 
錯誤的
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
硬體資訊
 a| 
節點的硬體資訊。
 a| 
xref:reference_element_api_hardwareinfo.adoc[硬體資訊]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetHardwareInfo",
   "params": {
   },
"id" : 1
}
----
