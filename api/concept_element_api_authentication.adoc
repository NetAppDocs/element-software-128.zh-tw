= API 驗證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 API 時，您可以透過在所有 API 請求中包含 HTTP 基本驗證標頭來對系統進行驗證。如果省略身份驗證訊息，系統將拒絕未經身份驗證的請求，並傳回 HTTP 401 回應。本系統支援基於TLS的HTTP基本驗證。

使用叢集管理員帳戶進行 API 驗證。
