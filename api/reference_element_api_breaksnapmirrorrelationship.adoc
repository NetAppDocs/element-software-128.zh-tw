= BreakSnapMirror關係
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element Web UI 使用 `BreakSnapMirrorRelationship`斷開SnapMirror連線的方法。當SnapMirror關係中斷時，目標磁碟區變成可讀寫且獨立，然後可能與來源磁碟區有所不同。你可以重新建立與…的關係 `ResyncSnapMirrorRelationship`API 方法。此方法需要ONTAP叢集可用。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
與 Element 儲存叢集通訊的遠端ONTAP儲存系統的端點 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
目的地捲
 a| 
SnapMirror關係中的目標磁碟區。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeInfo]
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法具有以下傳回值：

|===


| Name | 描述 | 類型 


 a| 
snapMirror關係
 a| 
包含有關已中斷的SnapMirror關係的資訊的物件。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror關係]

|===
