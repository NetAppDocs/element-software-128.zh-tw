= 啟用 BBC
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `EnableSsh`為單一節點啟用安全外殼（SSH）服務的方法。此方法不會影響叢集範圍內的 SSH 逾時持續時間，也不會使節點免於全域 SSH 逾時導致的 SSH 停用。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
已啟用
 a| 
此節點的 SSH 服務狀態。
 a| 
布林值

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "EnableSsh",
  "params": {
     },
  "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": true}
}
----
