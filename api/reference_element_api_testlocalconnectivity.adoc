= 測試本地連接
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `TestLocalConnectivity`用於 ping 活動叢集中每個節點的叢集 IP (CIP) 的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
細節
 a| 
本地活動叢集中每個節點的單獨 ping 回應時間。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
    "method": "TestLocalConnectivity",
    "params": {},
    "id": 1
}
----
