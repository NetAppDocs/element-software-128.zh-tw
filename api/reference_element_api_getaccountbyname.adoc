= 按名稱取得帳戶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `GetAccountByName`根據使用者名稱取得特定帳戶的詳細資訊。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
使用者名稱
 a| 
帳戶用戶名。
 a| 
細繩
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
帳戶
 a| 
帳戶詳情。
 a| 
xref:reference_element_api_account.adoc[帳戶]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetAccountByName",
   "params": {
     "username"  : "jimmyd"
   },
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
    "account": {
        "attributes": {},
        "username": "jimmyd",
        "targetSecret": "targetsecret",
        "volumes": [],
        "enableChap": true,
        "status": "active",
        "accountID": 1,
        "storageContainerID": "abcdef01-1234-5678-90ab-cdef01234567",
        "initiatorSecret": "initiatorsecret"
    }
}
----
