= 列出節點統計資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListNodeStats`查看儲存叢集中所有儲存節點的進階活動測量的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
節點統計
 a| 
儲存節點活動資訊。
 a| 
xref:reference_element_api_nodestats.adoc[節點統計]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListNodeStats",
   "params": {},
   "id" : 1
}
----
