= 刪除所有支援包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteAllSupportBundles`刪除所有使用以下方式產生的支援包的方法 `CreateSupportBundle`API 方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "DeleteAllSupportBundles",
   "params": {}
  },
  "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {}
 }
}
----
