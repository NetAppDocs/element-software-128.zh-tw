= 關閉
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `Shutdown`重啟或關閉叢集中節點的方法。您可以使用此方法關閉叢集中的單一節點、多個節點或所有節點。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
節點
 a| 
若要重新啟動或關閉的節點的 NodeID 清單。
 a| 
整數數組
 a| 
沒有任何
 a| 
是的



 a| 
選項
 a| 
針對該集群應採取的行動。可能的值：

* 重啟：重啟集群。
* 停止：執行完全斷電。

 a| 
細繩
 a| 
重啟
 a| 
不

|===


== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "Shutdown",
   "params": {
     "nodes": [
       2,
       3,
       4
     ],
     "option": "halt"
   },
   "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {
     "failed": [],
     "successful": [
          6
      [
   }
}
----
