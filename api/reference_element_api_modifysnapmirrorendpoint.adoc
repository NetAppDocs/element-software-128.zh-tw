= 修改快照鏡像端點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `ModifySnapMirrorEndpoint`更改SnapMirror端點的名稱和管理屬性的方法。



== 參數

此方法有以下輸入參數：

|===


| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
SnapMirrorEndpointID
 a| 
要修改的SnapMirror端點。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
管理IP
 a| 
ONTAP系統的新管理IP位址。
 a| 
細繩
 a| 
沒有任何
 a| 
不



 a| 
使用者名稱
 a| 
ONTAP系統的新管理使用者名稱。
 a| 
細繩
 a| 
沒有任何
 a| 
不



 a| 
密碼
 a| 
ONTAP系統的新管理密碼。
 a| 
細繩
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
snapMirrorEndpoint
 a| 
關於已修改的SnapMirror端點的資訊。
 a| 
xref:reference_element_api_snapmirrorendpoint.adoc[snapMirrorEndpoint]

|===
