= 列出虛擬卷任務
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVirtualVolumeTasks`取得系統中虛擬磁碟區任務清單的方法。



== 參數

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
虛擬磁碟區任務 ID
 a| 
要檢索資訊的虛擬磁碟區任務 ID 清單。如果省略此參數，則方法將傳回有關所有虛擬磁碟區任務的資訊。
 a| 
UUID 數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
任務
 a| 
描述叢集中虛擬磁碟區任務的物件清單。
 a| 
xref:reference_element_api_task_virtual_volumes.adoc[任務]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListVirtualVolumeTasks",
    "params": {
    },
    "id": 1
}
----
