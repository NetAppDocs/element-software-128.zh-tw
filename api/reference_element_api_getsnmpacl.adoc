= 取得 SNMP 存取控制列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetSnmpACL`取得叢集節點上目前 SNMP 存取權限的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
網路
 a| 
列出各個網路及其對叢集節點上執行的 SNMP 伺服器的存取權限類型。如果 SNMP v3 被停用，則存在此值。
 a| 
xref:reference_element_api_network_snmp.adoc[網路]大批



 a| 
usmUsers
 a| 
列出在叢集節點上執行的 SNMP 伺服器的使用者及其存取權限類型。如果啟用了 SNMP v3，則存在此值。
 a| 
xref:reference_element_api_usmuser.adoc[usmUser]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
"method": "GetSnmpACL",
"params": {},
"id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {
     "usmUsers"  : [
       {
          "name": "jdoe",
          "access": "rouser",
          "secLevel": "priv",
          "password": "mypassword",
          "passphrase": "mypassphrase",
       }
     ]
   }
}
----
