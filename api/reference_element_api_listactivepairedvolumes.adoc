= 列出活動配對卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListActivePairedVolumes`列出與某個卷配對的所有活動卷的方法。此方法傳回有關具有活動和待處理配對的交易量的資訊。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
卷
 a| 
配對卷的捲數資訊。
 a| 
xref:reference_element_api_volumepair.adoc[體積對]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListActivePairedVolumes",
   "params": {
      },
   "id" : 1
}
----
