= 取得Lldp配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetLldpConfig`取得儲存叢集中每個節點的連結層發現協定 (LLDP) 配置的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
lldp配置
 a| 
儲存叢集 LLDP 設定資訊。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetLldpConfig",
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
	"id": null,
		"result": {
			"lldpConfig": {
				"enableLldp": true,
				"enableMed": false,
				"enableOtherProtocols": true
		}
	}
}
----
