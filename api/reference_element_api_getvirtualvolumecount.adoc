= 取得虛擬卷計數
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetVirtualVolumeCount`取得系統中目前虛擬磁碟區數量的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
數數
 a| 
系統中目前虛擬卷的數量。
 a| 
整數

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "GetVirtualVolumeCount",
    "params": {
    },
    "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 1,
  "result": {
    "count": 5
  }
}
----
