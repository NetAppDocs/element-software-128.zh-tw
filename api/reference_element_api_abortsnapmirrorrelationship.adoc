= 中止快照鏡像關係
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `AbortSnapMirrorRelationship`停止已開始但尚未完成的SnapMirror傳輸的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
與 Element 儲存叢集通訊的遠端ONTAP儲存系統的端點 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
目的地捲
 a| 
SnapMirror關係中的目標磁碟區。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeInfo]
 a| 
沒有任何
 a| 
是的



 a| 
清除檢查點
 a| 
決定是否清除重啟檢查點。可能的值：

* 真的
* 錯誤的

 a| 
布林值
 a| 
錯誤的
 a| 
不

|===


== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
snapMirror關係
 a| 
包含有關已中止的SnapMirror關係的資訊的物件。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror關係]

|===
