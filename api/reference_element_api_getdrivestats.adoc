= 取得駕駛統計數據
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDriveStats`取得單一驅動器高級活動測量的方法。從將驅動器新增到叢集開始，數值是累積的。某些值是區塊磁碟機特有的。執行此方法時，將傳回區塊磁碟機或元資料磁碟機類型的統計資料。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
磁碟機ID
 a| 
請求所用磁碟機的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
驅動統計
 a| 
指定磁碟機 ID 的磁碟機活動資訊。
 a| 
xref:reference_element_api_drivestats.adoc[驅動統計]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetDriveStats",
   "params": {
     "driveID": 3
   },
   "id" : 1
}
----
