= 取得 BinAssignmentProperties
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetBinAssignmentProperties`用於從資料庫中檢索箱分配屬性的方法。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
特性
 a| 
詳細列出資料庫中所有目前倉庫分配的屬性。
 a| 
binAssignmentProperties 數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
    "method": "GetBinAssignmentProperties",
    "params": {
    },
    "id": 1
}
----
