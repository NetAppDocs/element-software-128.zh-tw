= 節點統計
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
nodeStats 物件包含節點的高階活動測量值。你可以使用 `GetNodeStats`和 `ListNodeStats`取得部分或全部 nodeStats 物件的 API 方法。
