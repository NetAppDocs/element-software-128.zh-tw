= 測試金鑰伺服器Kmip
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `TestKeyServerKmip`測試指定的金鑰管理互通協定 (KMIP) 金鑰伺服器是否可達且運作正常的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
密鑰伺服器ID
 a| 
要測試的KMIP金鑰伺服器的ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。如果沒有傳回錯誤，則測試被視為成功。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "TestKeyServerKmip",
  "params": {
    "keyServerID": 15
  },
"id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id": 1,
   "result":
      {}
    }
----
