= 列表實用程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListUtilities`列出可在節點上運行的操作的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
公用事業
 a| 
目前可在節點上執行的實用程式清單。
 a| 
字串數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListUtilities",
   "params": {},
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 1,
  "result": {
    "utilities": [
      "ResetDrives",
      "ResetNode",
      "RestartNetworking",
      "RestartServices",
      "CreateSupportBundle",
      "DeleteAllSupportBundles",
      "CreateClusterSupportBundle"
    ]
  }
}
----
