= 重置驅動器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ResetDrives`主動初始化磁碟機並刪除磁碟機上目前所有資料的方法。然後，該驅動器可以在現有節點中重複使用，也可以在升級後的節點中使用。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
驅動
 a| 
要重置的裝置名稱清單（不是磁碟機 ID）。
 a| 
細繩
 a| 
沒有任何
 a| 
是的



 a| 
力量
 a| 
設定為 true 可重置驅動器。
 a| 
布林值
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
細節
 a| 
正在重置的驅動器的詳細資訊。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ResetDrives",
   "params": {
      "drives" : "slot3",
      "force" : true
   },
   "id" : 1
}
----
