= 取得QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetQoSPolicy`從系統中取得特定 QoS 策略詳細資訊的方法。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
qosPolicyID
 a| 
要檢索的保單 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
服務品質策略
 a| 
所要求的服務品質策略詳情。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
	"method": "GetQoSPolicy",
    "params": {
    	"qosPolicyID": 2
    },
    "id": 1
}
----
