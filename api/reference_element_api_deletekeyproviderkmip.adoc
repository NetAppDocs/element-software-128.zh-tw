= 刪除金鑰提供者Kmip
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteKeyProviderKmip`刪除指定的非活動金鑰管理互通協定 (KMIP) 金鑰提供者的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
密鑰提供者ID
 a| 
要刪除的密鑰提供者的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。只要沒有錯誤，刪除操作就被認為是成功的。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "DeleteKeyProviderKmip",
  "params": {
    "keyProviderID": "1"
    },
"id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id": 1,
   "result":
      {}
}
----
