= 叢集 API 方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 軟體叢集 API 方法可讓您管理儲存叢集及其所屬節點的配置和拓撲。

某些叢集 API 方法在屬於叢集的節點上操作，或在已配置為加入叢集的節點上操作。您可以為新叢集或現有叢集新增節點。準備新增到叢集中的節點處於「待處理」狀態，這表示它們已經配置好，但尚未新增到叢集中。
