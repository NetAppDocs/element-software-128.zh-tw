= 列表卷統計
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStats`取得單一磁碟區、磁碟區清單或所有磁碟區（如果省略 volumeIDs 參數）的進階活動測量的方法。測量值從體積形成之初開始累積。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
包含虛擬卷
 a| 
預設情況下，回應中包含虛擬磁碟區。若要排除虛擬卷，請設定為 false。
 a| 
布林值
 a| 
真的
 a| 
不



 a| 
卷 ID
 a| 
可從中檢索活動資訊的磁碟區清單。
 a| 
整數數組
 a| 
不
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
銷量統計
 a| 
成交量活動資訊列表。
 a| 
xref:reference_element_api_volumestats.adoc[銷量統計]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListVolumeStats",
    "params": {
        "volumeIDs": [1]
    },
    "id": 1
}
----
