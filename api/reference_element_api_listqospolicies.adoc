= 列出QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListQoSPolicies`列出系統上所有 QoS 策略設定的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
服務品質策略
 a| 
列出每項QoS策略的詳細資訊。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 231,
  "method": "ListQoSPolicies",
  "params": {}
}
----
