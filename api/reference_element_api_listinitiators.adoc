= 列表發起者
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListInitiators`取得發起方 IQN 或全球連接埠名稱 (WWPN) 清單的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
發起者
 a| 
要檢索的發起方 ID 清單。您可以提供此參數或 startInitiatorID 參數，但不能同時提供兩者。
 a| 
整數數組
 a| 
沒有任何
 a| 
不



 a| 
啟動器ID
 a| 
清單開始的發起者 ID。您可以提供此參數或啟動器參數，但不能同時提供兩者。
 a| 
整數
 a| 
0
 a| 
不



 a| 
限制
 a| 
要傳回的啟動器物件的最大數量。
 a| 
整數
 a| 
（無限）
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
發起者
 a| 
發起人資訊列表。
 a| 
xref:reference_element_api_initiator.adoc[發起者]大批

|===


== 例外情況

此方法可能存在以下例外：

|===


| Name | 描述 


 a| 
xInvalidParameter
 a| 
如果在同一個方法呼叫中同時包含 startInitiatorID 和 initiators 參數，則會拋出例外。

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListInitiators",
   "params": {},
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 1,
  "result": {
    "initiators": [
      {
        "alias": "",
        "attributes": {},
        "initiatorID": 2,
        "initiatorName": "iqn.1993-08.org.debian:01:c84ffd71216",
        "volumeAccessGroups": [
          1
        ]
      }
    ]
  }
}
----
