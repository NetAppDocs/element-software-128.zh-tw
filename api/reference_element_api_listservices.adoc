= 清單服務
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListServices`列出叢集中正在運行的節點、磁碟機、目前軟體和其他服務的服務資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
服務
 a| 
在驅動器和節點上運行的服務。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListServices",
   "params": {},
   "id" : 1
}
----
