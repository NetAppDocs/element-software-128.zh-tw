= 創建公鑰/私鑰對
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `CreatePublicPrivateKeyPair`創建公鑰和私鑰的方法。您可以使用這些金鑰產生憑證簽署請求。每個儲存叢集只能使用一對密鑰。在使用此方法替換現有金鑰之前，請確保金鑰不再被任何提供者使用。

== 傳回值

此方法沒有傳回值。如果沒有錯誤，則密鑰建立被視為成功。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "CreatePublicPrivateKeyPair",
  "params": {
    "commonName": "Name",
    "country": "US",
    "emailAddress" : "email@domain.com"
   },
"id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id": 1,
   "result":
      {}
}
----
