= 測試遠端連線
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `TestRemoteConnectivity`ping 遠端叢集的每個節點並檢查遠端叢集資料庫連線的方法。必須將聚類結果成對才能使用此方法獲得有用的結果。如果遠端資料庫連線失敗，系統回應會列出異常情況。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
細節
 a| 
每個節點的單獨 ping 回應時間。
 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "TestRemoteConnectivity",
    "params": {
      "force": "true"
    },
    "id": 1
}
----
