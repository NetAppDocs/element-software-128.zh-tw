= snapMirrorAggregate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorAggregate 物件包含有關可用ONTAP聚合的信息，這些聚合是提供給磁碟區作為儲存的磁碟集合。您可以使用 ListSnapMirrorAggregates API 方法來取得此資訊。
