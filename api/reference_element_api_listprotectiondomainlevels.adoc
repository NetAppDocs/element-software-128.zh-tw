= 列表保護域級別
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListProtectionDomainLevels`列出儲存叢集的容錯性和彈性等級的方法。容錯等級表示叢集在發生故障時繼續讀取和寫入資料的能力，而復原等級表示儲存叢集從一次或多次故障中自動復原自身的能力。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
保護域級別
 a| 
列出了不同的保護域級別，每個級別都提供了儲存叢集的容錯性和復原能力資訊。
 a| 
xref:reference_element_api_protectiondomainlevel.adoc[保護域級別]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListProtectionDomainLevels",
  "params": {},
  "id" : 1
}
----
