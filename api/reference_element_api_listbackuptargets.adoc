= 列出備份目標
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListBackupTargets`取得所有已建立備份目標資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
備份目標
 a| 
每個備份目標傳回的物件。包含的物件：

* attributes：JSON 物件格式的名稱-值對清單。  （JSON 對象）
* backupTargetID：分配給備份目標的唯一識別碼。  （整數）
* 名稱：備份目標的名稱。  （細繩）

 a| 
JSON 物件

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListBackupTargets",
   "params": {},
   "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id": 1,
   "result": {
     "backupTargets": [
       {
        "attributes" : {},
        "backupTargetID" : 1,
        "name" : "mytargetbackup"
       }
    ]
  }
}
----
