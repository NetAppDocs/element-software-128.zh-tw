= 取得 IdP 驗證狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetIdpAuthenticationState`傳回有關使用第三方身分提供者進行身分驗證狀態的資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
已啟用
 a| 
指示是否啟用第三方身分提供者身份驗證。
 a| 
布林值

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "GetIdpAuthenticationState"
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "result": {"enabled": true}
}
----
