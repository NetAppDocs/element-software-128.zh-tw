= 列出協定端點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListProtocolEndpoints`用於檢索叢集中所有協定端點資訊的方法。協定端點控制對其關聯的虛擬磁碟區儲存容器的存取。



== 參數

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
協定端點 ID
 a| 
要檢索資訊的協定端點 ID 清單。如果省略此參數，則方法將傳回有關所有協定端點的資訊。
 a| 
協定端點 ID UUID 數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
協定端點
 a| 
包含系統中每個協定端點資訊的物件清單。
 a| 
xref:reference_element_api_protocolendpoint.adoc[協定端點]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 1,
  "method": "ListProtocolEndpoints",
  "params": {}
}
----
