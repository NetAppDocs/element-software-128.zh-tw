= 取得遠端日誌主機
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetRemoteLoggingHosts`取得目前日誌伺服器清單的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
遠端主機
 a| 
列出已設定為接收轉送日誌資訊的主機的 IP 位址和連接埠資訊。
 a| 
xref:reference_element_api_loggingserver.adoc[日誌伺服器]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "id": 3386609,
   "method": "GetRemoteLoggingHosts",
   "params": {}
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id": 3386609,
   "result": {
      "remoteHosts": [
         {
           "host": "172.16.1.20",
           "port": 10514
         },
         {
           "host": "172.16.1.25"
         }
      ]
   }
}
----
