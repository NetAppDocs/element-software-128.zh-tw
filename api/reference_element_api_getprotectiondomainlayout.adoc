= 取得保護域佈局
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetProtectionDomainLayout`傳回叢集所有保護域資訊的方法，包括每個節點所在的機箱和自訂保護域。



== 範圍

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
保護域佈局
 a| 
節點列表，每個節點及其關聯的保護域。
 a| 
JSON 列表link:reference_element_api_nodeprotectiondomains.html["節點保護域"]物體。

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "GetProtectionDomainLayout",
  "params": {},
  "id" : 1
}
----
