= 取得驅動硬體資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDriveHardwareInfo`取得給定驅動器所有硬體資訊的方法。這通常包括製造商、供應商、版本和其他相關的硬體識別資訊。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
磁碟機ID
 a| 
請求所用磁碟機的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
結果
 a| 
傳回指定磁碟機 ID 的硬體資訊。
 a| 
xref:reference_element_api_hardwareinfo.adoc[硬體資訊]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetDriveHardwareInfo",
   "params": {
     "driveID": 5
   },
   "id" : 100
}
----
