= 批量作業列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListBulkVolumeJobs`取得系統中正在發生的每個大容量磁碟區讀取或寫入操作的資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
批量作業
 a| 
每個大批量作業的資訊數組。
 a| 
xref:reference_element_api_bulkvolumejob.adoc[批量作業]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
