= 列出卷 QoS 直方圖
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeQoSHistograms`產生一個或多個磁碟區的磁碟區 QoS 使用情況直方圖的方法。這使您能夠更好地了解流量是如何使用 QoS 的。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
卷 ID
 a| 
一個可選的磁碟區 ID 列表，用於指定應該為哪些磁碟區產生 QoS 直方圖。
 a| 
整數數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
qos直方圖
 a| 
描述一個或多個磁碟區的磁碟區使用情況的物件清單。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
	"method": "ListVolumeQoSHistograms",
		"params": {
			"volumeIDs": [1]
		},
	"id": 1
}
----
