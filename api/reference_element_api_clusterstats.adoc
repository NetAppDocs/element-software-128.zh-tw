= 集群統計
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
clusterStats 物件包含叢集的統計資料。物件中包含的許多與體積相關的統計資訊都是對集群中所有體積進行平均所得的。您可以使用 GetClusterStats 方法檢索叢集的此資訊。
