= 驅動統計
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
driveStats 物件包含單一磁碟機的進階活動測量值。您可以使用 API 方法檢索測量資訊。 `GetDriveStats` 。
