細繩
 a| 
沒有任何
 a| 
是的



 a| 
使用者名稱
 a| 
ONTAP系統的管理使用者名稱。
 a| 
細繩
 a| 
沒有任何
 a| 
是的

密碼

ONTAP系統的管理密碼。
 a| 
細繩
 a| 
沒有任何
 a| 
是的

|===
