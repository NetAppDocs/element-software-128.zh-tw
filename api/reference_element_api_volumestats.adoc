= 銷量統計
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
volumeStats 物件包含單一磁碟區的統計資料。



== 對象成員

您可以使用以下方法取得部分或全部磁碟區的 volumeStats 物件：

* xref:reference_element_api_getvolumestats.adoc[獲取成交量統計信息]
* xref:reference_element_api_listvolumestatsbyaccount.adoc[按帳戶列出交易量統計信息]
* xref:reference_element_api_listvolumestatsbyvolume.adoc[按成交量列出成交量統計數據]
* xref:reference_element_api_listvolumestatsbyvolumeaccessgroup.adoc[按卷訪問組列出卷統計信息]

 a| 
卷的 I/O 使用情況統計資料。從 Element 12.8 版本開始可用。  sliceIopsStats 的可能值：

* 大型統計資料：在較長時間段內（通常為最近 24 小時）測量的 I/O 容量統計資料。
* smallStatistics：在較短時間內（通常是最近一小時）測量的 I/O 流量統計資料。


largeStatistics 和 smallStatistics 的可能取值：

* averageReadIOPS：卷的平均讀取 IOPS。
* averageTotalIops：卷的平均總（讀取+寫入）IOPS。
* averageWriteIOPS：卷的平均寫入 IOPS。
* nSamples：統計計算中包含的樣本數。
* peakReadIOPS：在統計間隔內觀察到的最大讀取IOPS。
* peakTotalIOPS：在統計區間內觀察到最大的總IOPS。
* peakWriteIOPS：在統計間隔內觀察到的最大寫入IOPS。
* sliceID：磁碟區 ID 或切片 ID
