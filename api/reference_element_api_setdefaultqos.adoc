= 設定預設服務質量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `SetDefaultQoS`配置磁碟區的預設服務品質 (QoS) 值（以每秒輸入和輸出數 (IOPS) 衡量）的方法。

== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "SetDefaultQoS",
    "params": {
        "burstIOPS":8000,
        "maxIOPS":1000,
        "minIOPS":200
    },
    "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
    "id":1,
    "result": {
        "burstIOPS":8000,
        "maxIOPS":1000,
        "minIOPS":200
    ​}
}
