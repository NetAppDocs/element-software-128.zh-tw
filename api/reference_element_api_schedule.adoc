= 行程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
計劃對象包含有關為自動建立磁碟區快照而建立的計劃的資訊。您可以使用以下方式擷取所有班次的班次資訊： `ListSchedules` API 方法。



== 對象成員

該物件包含以下成員：

指示調度程序下次執行時是否會執行該調度程序。如果為真，則計劃將在調度程式下次啟動時運行，然後該值將設定回假。可能的值：

* 錯誤的

 a|

 a|

 a| 
日程資訊
 a| 
包括為計劃賦予的唯一名稱、建立的快照的保留期限以及建立快照的磁碟區的磁碟區 ID。有效值：

* `enableRemoteReplication`：指示是否應將快照包含在遠端複製中。  （布林值）
* `ensureSerialCreation`：指定如果先前的快照複製正在進行中，是否應允許建立新的快照。  （布林值）
* `name`要使用的快照名稱。  （細繩）
* `retention`快照保留的時間。根據時間不同，它會以以下格式之一顯示：
+
** `fifo`快照依先進先出 (FIFO) 原則保留。如果快照為空，則永久保留。  （細繩）
** 時:分:秒


* `volumeID`：要包含在快照中的磁碟區的 ID。  （整數）
* `volumes`：要包含在群組快照中的磁碟區 ID 清單。  （整數數組）
