= 建立QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `CreateQoSPolicy`建立一個 QoSPolicy 物件的方法，您可以在建立或修改磁碟區時將其套用到磁碟區。  QoS策略具有唯一的ID、名稱和QoS設定。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
姓名
 a| 
QoS策略的名稱；例如，金牌、白金或銀牌。
 a| 
細繩
 a| 
沒有任何
 a| 
是的



 a| 
服務品質
 a| 
此策略所代表的 QoS 設定。
 a| 
xref:reference_element_api_qos.adoc[服務品質]
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
服務品質策略
 a| 
新建立的QoSPolicy物件。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 68,
  "method": "CreateQoSPolicy",
  "params": {
    "name": "bronze",
    "qos": {
      "minIOPS": 50,
      "maxIOPS": 15000,
      "burstIOPS": 15000
    }
  }
}
----
