= 取得網路配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNetworkConfig`顯示節點網路配置資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
網路
 a| 
節點的每個網路介面的網路連線類型和目前設定。
 a| 
xref:reference_element_api_network_all_interfaces.adoc[網路（所有介面）]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetNetworkConfig",
   "params": {},
   "id" : 1
}
----
