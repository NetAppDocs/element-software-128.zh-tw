= 清單卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumes`取得集群中卷列表的方法。您可以使用可用參數指定要在清單中傳回的磁碟區。

 a|

 a| 
沒有任何
 a| 
不



 a|

 a|

 a| 
布林值
 a| 
真的
 a| 
不



 a|

 a| 
傳回已配對或未配對的磁碟區。可能的值：

* true：傳回所有配對磁碟區。
* false：傳回所有未配對的磁碟區。

 a|

 a|

== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
卷
 a| 
卷冊列表。
 a| 
xref:reference_element_api_volume.adoc[體積]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListVolumes",
    "params": {
        "volumeIDs": [1],
        "volumeStatus": "active",
        "isPaired": "false"
    },
    "id": 1
}
----
