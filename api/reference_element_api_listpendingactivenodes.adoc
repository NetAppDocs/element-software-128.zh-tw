= 列出待處理活動節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListPendingActiveNodes`列出叢集中處於 PendingActive 狀態（介於 pending 和 active 狀態之間）的節點的方法。處於此狀態的節點將被恢復到出廠設定。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
待處理活動節點
 a| 
列出系統中所有 PendingActive 節點的詳細資訊的物件清單。
 a| 
xref:reference_element_api_pendingactivenode.adoc[待處理活動節點]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListPendingActiveNodes",
   "params": {},
   "id" : 1
}
----
