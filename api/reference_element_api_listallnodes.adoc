= 列出所有節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListAllNodes`列出叢集中活動節點和待處理節點的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
節點
 a| 
描述叢集中活動節點的物件清單。
 a| 
xref:reference_element_api_node.adoc[節點]



 a| 
待處理活動節點
 a| 
描述叢集中待激活節點的物件列表。
 a| 
xref:reference_element_api_pendingactivenode.adoc[待處理活動節點]大批



 a| 
待處理節點
 a| 
描述叢集待處理節點的物件清單。
 a| 
xref:reference_element_api_pendingnode.adoc[待處理節點]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListAllNodes",
   "params": {},
   "id" : 1
}
----
