= 清單驅動器硬體
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListDriveHardware`列出連接到節點的所有磁碟機的方法。當用於單一節點時，此方法會傳回驅動器硬體資訊。在叢集主節點 MVIP 上使用時，此方法傳回所有節點上所有磁碟機的資訊。



== 參數


NOTE: 方法回應中的「securitySupported」: true 行並不表示磁碟機能夠加密；僅表示可以查詢安全狀態。如果節點類型型號以“-NE”結尾，則啟用這些磁碟機的安全功能的命令將會失敗。

此方法具有以下參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
力量
 a| 
設定為 true 可在所有節點上執行此方法。
 a| 
布林值
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
驅動硬體
 a| 
返回節點的驅動器硬體資訊。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
"method": "ListDriveHardware",
"params": {},
"id" : 1
}
----
