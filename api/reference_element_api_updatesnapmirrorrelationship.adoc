= 更新SnapMirror關係
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `UpdateSnapMirrorRelationship`使SnapMirror關係中的目標磁碟區成為來源磁碟區的最新鏡像的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
與 Element 儲存叢集通訊的遠端ONTAP儲存系統的端點 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
目的地捲
 a| 
SnapMirror關係中的目標磁碟區。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeinfo]
 a| 
沒有任何
 a| 
是的



 a| 
最大轉帳費率
 a| 
指定磁碟區之間的最大資料傳輸速率，單位為千字節每秒。預設值 0 表示無限制，允許SnapMirror關係充分利用可用的網路頻寬。
 a| 
整數
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
snapMirror關係
 a| 
包含更新後的SnapMirror關係資訊的物件。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror關係]

|===
