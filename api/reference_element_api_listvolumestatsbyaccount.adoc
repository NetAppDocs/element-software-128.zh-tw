= 按帳戶列出交易量統計信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStatsByAccount`列出每個帳戶的高級交易量指標的方法。數值是帳戶所擁有的所有交易量的總和。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
包含虛擬卷
 a| 
預設情況下，回應中包含虛擬磁碟區。若要排除虛擬卷，請設定為 false。
 a| 
布林值
 a| 
真的
 a| 
不



 a| 
帳戶
 a| 
若要傳回交易量統計資料的帳戶 ID 清單。如果省略，則傳回所有帳戶的統計資料。
 a| 
整數數組
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
銷量統計
 a| 
每個帳戶的捲活動資訊清單。 *注意：*每個條目的 volumeID 成員均為 0，因為這些值表示帳戶擁有的所有磁碟區的總和。
 a| 
xref:reference_element_api_volumestats.adoc[銷量統計]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
"method": "ListVolumeStatsByAccount",
  "params": {"accounts": [3]},
  "id": 1
}
----
