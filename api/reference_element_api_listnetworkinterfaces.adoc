= 列出網路介面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListNetworkInterfaces`列出節點上每個網路介面資訊的方法。此 API 方法旨在用於單一節點；存取單一節點需要使用者 ID 和密碼身份驗證。但是，如果在方法呼叫中將參數 force 的值設為 true，則可以在叢集上使用此方法。在叢集上使用該參數時，將列出所有介面。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
力量
 a| 
可能的值：

* true：傳回叢集中所有網路介面的資訊。
* false：未回傳任何訊息。

 a| 
布林值
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
介面
 a| 
儲存節點（或整個儲存集群，如果 force = ）的每個網路介面的配置資訊列表 `true`）。
 a| 
xref:reference_element_api_networkinterface.adoc[網路介面]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
"method": "ListNetworkInterfaces",
"params": {},
"id" : 1
}
----
