= 列出日程表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListSchedules`取得所有已建立的計劃快照的資訊。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
日程安排
 a| 
集群上目前所有調度程序的列表。
 a| 
xref:reference_element_api_schedule.adoc[行程]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "ListSchedules",
    "params": {},
"id": 1
}
----
