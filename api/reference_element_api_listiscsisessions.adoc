= 列出 ISCSI 會話
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListISCSISessions`列出叢集中磁碟區的 iSCSI 連線資訊的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
會議
 a| 
每個 iSCSI 會話的資訊。
 a| 
xref:reference_element_api_session_iscsi.adoc[會議]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListISCSISessions",
   "params": {},
   "id" : 1
}
----
