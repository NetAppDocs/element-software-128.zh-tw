= ListVolumesForAccount
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumesForAccount`列出帳戶的活動捲和（待刪除的）卷的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
包含虛擬卷
 a| 
預設情況下，回應中包含虛擬磁碟區。若要排除虛擬卷，請設定為 false。
 a| 
布林值
 a| 
真的
 a| 
不



 a| 
帳戶ID
 a| 
傳回此帳戶 ID 擁有的所有磁碟區。
 a| 
整數
 a| 
不
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
卷
 a| 
卷資訊列表。
 a| 
xref:reference_element_api_volume.adoc[體積]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListVolumesForAccount",
   "params": {
      "accountID"  : 1
   },
   "id" : 1
}
----
