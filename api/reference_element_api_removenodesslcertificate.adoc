= 移除節點SSL憑證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RemoveNodeSSLCertificate`刪除管理節點的使用者 SSL 憑證和私鑰的方法。刪除憑證和私鑰後，管理節點將配置為使用預設憑證和私鑰。



== 參數


NOTE: 必須對管理節點呼叫此方法。例如：

[listing]
----
https://<management node IP>:442/json-rpc/10.0
----
此方法沒有輸入參數。



== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
    "method" : "RemoveNodeSSLCertificate",
    "params" : {},
    "id" : 3
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
    "id" : 3,
    "result" : {}
}
----
