集群物件包含節點用於與集群通訊的資訊。您可以使用 GetClusterConfig API 方法來檢索此資訊。



== 對象成員

該物件包含以下成員：

|===
| Name | 描述 | 類型

 a|

 a|

 a|

 a|

 a|

 a| 
字串數組



 a| 
fipsDriveConfiguration
 a| 
指示節點是否支援 FIPS 140-2 認證的驅動器。
 a| 
布林值



 a| 
mipi
 a| 
用於節點管理的網路介面。
 a| 
細繩



 a| 
姓名
 a| 
群集名稱。

細繩



 a| 
節點ID
 a| 
叢集中節點的節點 ID。
 a|

 a|

 a|

 a| 
角色
 a| 
確定節點的角色。
 a| 
整數



 a| 
sipi
 a| 
用於儲存流量的網路介面。
 a| 
細繩



 a| 
狀態
 a| 
節點的當前狀態。可能的值：

* 可用：該節點尚未配置叢集名稱。
* 待新增：該節點正在等待加入指定的集群，可以新增。
* 活動節點：此節點是叢集的活動成員，不能新增到其他叢集。
* 待啟動：該節點目前正在恢復到出廠軟體映像，尚未成為叢集的活動成員。完成後，它將變為活動狀態。

 a| 
細繩



 a| 
版本
 a| 
節點上執行的軟體版本。
 a| 
細繩

|===
