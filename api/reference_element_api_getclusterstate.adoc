= 取得集群狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetClusterState`用於指示節點是否屬於叢集的 API 方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有以下傳回值：

|===
| Name | 描述 | 類型 


 a| 
簇
 a| 
群集名稱。
 a| 
細繩



 a| 
狀態
 a| 
* 可用：節點尚未配置叢集名稱。
* 待處理：該節點已等待加入指定的集群，可以新增。
* 活動節點：此節點是叢集的活躍成員，不能新增到其他叢集。

 a| 
細繩

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetClusterState",
   "params": {},
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
   "id" : 1,
   "result" :
      "cluster" :  "Cluster101"
      "state" : "Active"
}
----
