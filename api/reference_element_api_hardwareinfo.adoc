= 硬體資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
hardwareInfo 物件包含叢集中每個節點的硬體和狀態的詳細資訊。您可以使用以下方式檢索此資訊： `GetHardwareInfo` API 方法。
