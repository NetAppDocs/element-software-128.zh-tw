= 重啟網絡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RestartNetworking`重啟節點上的網路服務的方法。

警告：

此方法會重新啟動節點上的所有網路服務，導致網路連線暫時中斷。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
力量
 a| 
設定為 true 可重啟節點上的網路服務。
 a| 
布林值
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "RestartNetworking",
   "params": {
              "force" : true
             },
   "id" : 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{ "id" : 1,
  "result" : {}
}
----
