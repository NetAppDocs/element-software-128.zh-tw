= 取得 IPMI 訊息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetIpmiInfo`用於顯示系統監控的節點風扇、進氣和排氣溫度以及電源的感測器（物件）的詳細報告的方法。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
感應器
 a| 
節點內每個感測器的詳細資訊。
 a| 
JSON 物件數組

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "GetIpmiInfo",
  "params": {},
  "id" : 1
}
----
