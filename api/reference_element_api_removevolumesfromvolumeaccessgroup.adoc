= 從卷宗訪問群組移除卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RemoveVolumesFromVolumeAccessGroup`從指定的磁碟區存取群組中移除磁碟區的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
卷訪問群組ID
 a| 
若要從中刪除磁碟區的 VolumeAccessGroupID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
卷
 a| 
要從磁碟區存取群組中移除的磁碟區的 VolumeID。
 a| 
整數數組
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
卷訪問群組
 a| 
包含有關新修改的磁碟區存取群組資訊的物件。
 a| 
xref:reference_element_api_volumeaccessgroup.adoc[卷訪問群組]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "RemoveVolumesFromVolumeAccessGroup",
   "params": {
      "volumeAccessGroupID": 96,
      "volumes": [1,2]
   },
   "id": 1
}
----
