= 從磁碟區存取群組中刪除發起程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RemoveInitiatorsFromVolumeAccessGroup`從指定的磁碟區存取群組中移除發起程序的方法。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
卷訪問群組ID
 a| 
從中移除發起程序的磁碟區存取群組的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
發起者
 a| 
若要包含在磁碟區存取群組中的發起程序 ID 或名稱（IQN 和 WWPN）清單。如果傳遞一個啟動器名稱列表，則當啟動器尚不存在時，將建立這些啟動器。如果傳遞一個發起者 ID 列表，則如果其中任何發起者尚不存在，則該方法將傳回錯誤。傳遞發起者名稱已被棄用；應盡可能使用發起者 ID。
 a| 
整數數組（建議）或字串數組（已棄用）
 a| 
沒有任何
 a| 
不



 a| 
刪除孤兒發起者
 a| 
指定在從磁碟區存取群組中刪除啟動器物件後是否將其刪除。可能的值：

* true：從磁碟區存取群組中移除啟動器物件後，將其刪除。
* false：從磁碟區存取群組中刪除啟動器物件後，請勿將其刪除。這是預設值。

 a| 
布林值
 a| 
錯誤的
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
卷訪問群組
 a| 
包含有關新修改的磁碟區存取群組資訊的物件。
 a| 
xref:reference_element_api_volumeaccessgroup.adoc[卷訪問群組]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 13171,
  "method": "RemoveInitiatorsFromVolumeAccessGroup",
  "params": {
    "initiators": [114,115],
    "volumeAccessGroupID": 96
  }
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 13171,
  "result": {
    "volumeAccessGroup": {
      "attributes": {},
      "deletedVolumes": [
        327
      ],
      "initiatorIDs": [],
      "initiators": [],
      "name": "test",
      "volumeAccessGroupID": 96,
      "volumes": [
        346
      ]
    }
  }
}
----
