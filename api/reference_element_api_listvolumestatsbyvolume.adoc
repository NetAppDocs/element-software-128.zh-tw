= 按成交量列出成交量統計數據
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStatsByVolume`按卷列出每卷的高級活動測量值的方法。此數值自卷創建以來不斷累積。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
包含虛擬卷
 a| 
預設情況下，回應中包含虛擬磁碟區。若要排除虛擬卷，請設定為 false。
 a| 
布林值
 a| 
真的
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
銷量統計
 a| 
成交量活動資訊列表。
 a| 
xref:reference_element_api_volumestats.adoc[銷量統計]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListVolumeStatsByVolume",
   "params": {},
   "id" : 1
}
----
