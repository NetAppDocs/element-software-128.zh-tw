= 網路（乙太網路介面）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
網路（乙太網路介面）物件包含各個乙太網路介面的設定資訊。你可以使用 `GetConfig`和 `GetNetworkConfig`獲取儲存節點此資訊的方法。



== 對象成員

該物件包含以下成員：

|===


| Name | 描述 | 類型 


 a| 
保稅經理
 a| 
指定此實體介面已作為綁定從設備加入到哪個綁定介面。
 a| 
細繩



 a| 
家庭
 a| 
介面配置使用的位址族。目前支援 IPv4 的「inet」協定。
 a| 
細繩



 a| 
mac位址
 a| 
指派給介面並被網路觀察到的實際 MAC 位址。
 a| 
細繩



 a| 
macAddressPermanent
 a| 
製造商指派給介面的不可更改的 MAC 位址。
 a| 
細繩



 a| 
方法
 a| 
用於配置介面的方法。可能的值：

* loopback：用於定義 IPv4 回環介面。
* 手動：用於定義不會自動配置的介面。
* dhcp：可用於透過 DHCP 取得 IP 位址。
* 靜態：用於定義具有靜態分配的 IPv4 位址的乙太網路介面。

 a| 
細繩



 a| 
地位
 a| 
介面狀態。可能的值：

* 向下：介面處於非活動狀態。
* 向上：介面已準備就緒，但沒有連結。
* 運作正常：介面已準備就緒，連線已建立。

 a| 
細繩



 a| 
upAndRunning
 a| 
指示介面是否已準備就緒並具有連結。
 a| 
布林值

|===
