= 列出SnapMirror網路接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體的Web使用者介面使用 `ListSnapMirrorNetworkInterfaces`列出遠端ONTAP系統上所有可用SnapMirror介面的方法。



== 範圍

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
snapMirrorEndpointID
 a| 
僅傳回與指定端點 ID 關聯的網路介面。如果沒有提供端點 ID，系統將列出所有已知SnapMirror端點的介面。
 a| 
整數
 a| 
沒有任何
 a| 
不



 a| 
介面角色
 a| 
僅列出承擔指定角色的網路介面。
 a| 
細繩
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
snapMirrorNetworkInterfaces
 a| 
遠端ONTAP儲存系統上可用的SnapMirror網路介面清單。
 a| 
xref:reference_element_api_snapmirrornetworkinterface.adoc[snapMirror網路介面]大批

|===
