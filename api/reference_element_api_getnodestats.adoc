= 取得節點統計資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNodeStats`取得單一節點的高階活動測量值的方法。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
節點ID
 a| 
指定要傳回統計資訊的節點的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
節點統計
 a| 
節點活動資訊。
 a| 
xref:reference_element_api_nodestats.adoc[節點統計]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "GetNodeStats",
   "params": {
     "nodeID": 5
   },
   "id": 1
}
----
