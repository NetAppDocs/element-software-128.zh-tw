= 網路（綁定介面）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
網路（綁定介面）物件包含儲存節點上綁定網路介面的設定資訊。你可以使用 `GetConfig`和 `GetNetworkConfig`獲取儲存節點此資訊的方法。



== 對象成員

該物件包含以下成員：

|===


| Name | 描述 | 類型 


 a| 
地址
 a| 
指派給節點上此介面的 IPv4 位址。
 a| 
細繩



 a| 
addressV6
 a| 
指派給節點上 Bond1G 介面的 IPv6 管理位址。
 a| 
細繩



 a| 
債券發行延遲
 a| 
偵測到連結故障後，停用從設備前需要等待的時間（以毫秒為單位）。
 a| 
細繩



 a| 
bond-fail_over_mac
 a| 
網路介面的MAC位址配置。
 a| 
細繩



 a| 
邦德-米蒙
 a| 
以毫秒為單位，檢查 MII 連結狀態是否有連結故障的頻率。
 a| 
細繩



 a| 
鍵合模式
 a| 
鍵合模式。可能的值：

* 主動/被動（預設）
* 艾爾布
* LACP（建議）

 a| 
細繩



 a| 
債券主選
 a| 
指定何時選擇主綁定從設備作為活動從設備。可能的值：

* 總是
* 更好的
* 失敗

 a| 
細繩



 a| 
奴隸
 a| 
綁定的從介面列表。
 a| 
細繩



 a| 
鍵結-lacp_rate
 a| 
當債券模式為 LACP 時，利率可能會變為以下幾種之一：

* LACP 快速（預設）
* LACP慢速

 a| 
細繩



 a| 
債券發行延遲
 a| 
偵測到連結後，在啟用從設備之前等待的時間（以毫秒為單位）。
 a| 
細繩



 a| 
DNS 名稱伺服器
 a| 
用於網域名稱服務的地址列表，以逗號或空格分隔。
 a| 
細繩



 a| 
DNS搜尋
 a| 
以空格或逗號分隔的 DNS 搜尋網域清單。
 a| 
細繩



 a| 
家庭
 a| 
介面配置使用的位址族。目前支援 IPv4 的「inet」協定。
 a| 
細繩



 a| 
閘道
 a| 
用於從本機網路傳送流量的 IPv4 路由器網路位址。
 a| 
細繩



 a| 
gatewayV6
 a| 
用於從本地 Bond1G 網路傳送流量的 IPv6 路由器網路位址。
 a| 
細繩



 a| 
ipV6PrefixLength
 a| 
Bond1G 網路上 IPv6 流量的「net」類型靜態路由的子網路前綴長度。
 a| 
細繩



 a| 
mac位址
 a| 
指派給介面並被網路觀察到的實際 MAC 位址。
 a| 
細繩



 a| 
macAddressPermanent
 a| 
製造商指派給介面的不可更改的 MAC 位址。
 a| 
細繩



 a| 
方法
 a| 
用於配置介面的方法。可能的值：

* 環回介面：用於定義 IPv4 環回介面。
* 手動：用於定義不會自動配置的介面。
* dhcp：可用於透過 DHCP 取得 IP 位址。
* 靜態：用於定義具有靜態分配的 IPv4 位址的乙太網路介面。

 a| 
細繩



 a| 
行動通訊技術
 a| 
介面可以傳輸的最大資料包大小（以位元組為單位）。必須大於或等於 1500；最高支援 9000。
 a| 
細繩



 a| 
網路遮罩
 a| 
指定介面子網路的位元遮罩。
 a| 
細繩



 a| 
網路
 a| 
指示基於子網路遮罩的 IP 位址範圍的起始位置。
 a| 
細繩



 a| 
路線
 a| 
以逗號分隔的路由字串數組，要將其套用到路由表中。
 a| 
字串數組



 a| 
地位
 a| 
介面狀態。可能的值：

* 向下：介面處於非活動狀態。
* 向上：介面已準備就緒，但沒有連結。
* 運作正常：介面已準備就緒，連線已建立。

 a| 
細繩



 a| 
對稱路由規則
 a| 
節點上配置的對稱路由規則。
 a| 
字串數組



 a| 
upAndRunning
 a| 
指示介面是否已準備就緒並具有連結。
 a| 
布林值



 a| 
虛擬網路標籤
 a| 
介面的虛擬網路識別碼（VLAN 標籤）。
 a| 
細繩

|===
