= 列出群集管理員
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListClusterAdmins`傳回集群所有集群管理員清單的方法。

可以有多個叢集管理員帳戶，每個帳戶具有不同的權限等級。系統中只能有一個主集群管理員。主集群管理員是建立集群時建立的管理員。在叢集上設定 LDAP 系統時，也可以建立 LDAP 管理員。



== 參數

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
顯示隱藏
 a| 
顯示隱藏的叢集管理員用戶，例如 SNMP 管理員。
 a| 
布林值
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法傳回以下值：

|===


| Name | 描述 | 類型 


 a| 
集群管理員
 a| 
集群中所有集群管理員和 LDAP 管理員的資訊。
 a| 
xref:reference_element_api_clusteradmin.adoc[集群管理員]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ListClusterAdmins",
   "params": {},
    "showHidden": true
   "id" : 1
}
----
