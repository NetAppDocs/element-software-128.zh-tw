= 錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果在方法呼叫期間發生錯誤，則錯誤物件包含錯誤代碼和訊息。所有系統產生的錯誤錯誤代碼均為 500。



== 對象成員

該物件包含以下成員：

|===
| Name | 描述 | 類型 


 a| 
程式碼
 a| 
用於識別錯誤的數字代碼。所有系統產生的錯誤均回傳代碼 500。
 a| 
整數



 a| 
姓名
 a| 
用於標識所發生的特定錯誤的唯一識別碼。每種方法都會傳回一組已記錄的錯誤，但您也應該做好處理無法辨識的錯誤準備。
 a| 
細繩



 a| 
訊息
 a| 
錯誤描述，可能包含更多細節。
 a| 
細繩

|===
