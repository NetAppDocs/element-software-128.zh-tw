= 設定SSL憑證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `SetSSLCertificate`為叢集中的儲存節點設定使用者 SSL 憑證和私鑰的方法。


NOTE: 使用 API 後，必須重新啟動管理節點。



== 參數

此方法有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
證書
 a| 
證書的 PEM 編碼文字版本。 *注意：*設定節點或叢集憑證時，憑證必須包含 serverAuth 的 extendedKeyUsage 擴充功能。此擴充功能可使憑證在常見的作業系統和瀏覽器上無錯誤地使用。如果擴充功能不存在，API 將拒絕該證書，認為其無效。
 a| 
細繩
 a| 
沒有任何
 a| 
是的



 a| 
私鑰
 a| 
私鑰的 PEM 編碼文字版本。
 a| 
細繩
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。
