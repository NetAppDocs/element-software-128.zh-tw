= 重置網路配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ResetNetworkConfig`用於協助解決單一節點網路配置問題的方法。此方法將單一節點的網路配置重設為出廠預設值。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
   "method": "ResetNetworkConfig",
   "params": {},
   "id" : 1
}
----


== 回應範例

此方法不回傳響應。
