= 修改QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ModifyQoSPolicy`修改系統上現有QoS策略的方法。



== 參數

此方法有以下輸入參數：

|===


| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
qosPolicyID
 a| 
要修改的策略的 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的



 a| 
姓名
 a| 
如果提供，則 QoS 策略的名稱（例如金牌、白金、銀牌）將更改為該值。
 a| 
細繩
 a| 
沒有任何
 a| 
不



 a| 
服務品質
 a| 
如果提供了這些設置，則此策略的 QoS 設定將變更為這些設置。您可以提供部分 QoS 值，並且只更改部分 QoS 設定。
 a| 
QoS對象
 a| 
沒有任何
 a| 
不

|===


== 傳回值

此方法具有以下傳回值：

|===


| Name | 描述 | 類型 


 a| 
服務品質策略
 a| 
新修改的QoS策略詳情。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "id": 1950,
  "method": "ModifyQoSPolicy",
  "params": {
    "qosPolicyID": 2,
    "qos": {
      "minIOPS": 51,
      "maxIOPS": 15002,
      "burstIOPS": 15002
    }
  }
}
----
