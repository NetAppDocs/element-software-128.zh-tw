= 取得日程安排
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `GetSchedule`取得有關計劃快照的資訊。

如果系統中有很多快照計劃，您就可以查看有關特定計劃的資訊。您也可以透過在 scheduleID 參數中指定其他 ID 來使用此方法檢索有關多個日程安排的資訊。



== 範圍

此方法具有以下輸入參數：

|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
計劃ID
 a| 
要顯示的日程表或多個日程表的唯一 ID。
 a| 
整數
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法傳回以下值：

|===
| Name | 描述 | 類型 


 a| 
行程
 a| 
日程屬性數組。
 a| 
xref:reference_element_api_schedule.adoc[行程]大批

|===


== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
"method": "GetSchedule",
  "params": {
              "scheduleID" : 2
      },
   "id" : 1
}
----
