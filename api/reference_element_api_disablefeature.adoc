= 停用功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DisableFeature`禁用方法 `VolumeLoadBalanceOnActualIOPS`特徵。 `VolumeLoadBalanceOnActualIOPS`預設已禁用。

[NOTE]
====
如果嘗試停用其他功能（例如 VVols），則會出現以下錯誤訊息：

[listing]
----
{"error":{"code":500,"message":"VolumeLoadBalanceOnActualIOPS is the only feature that can be disabled","name":"xAPINotPermitted"},"id":null}
----
====


== 範圍

此方法有以下輸入參數。

[cols="1a,3a,1a,1a,1a"]
|===
| Name | 描述 | 類型 | 預設值 | 必需的 


 a| 
特徵
 a| 
禁用集群功能。可能的值：

* `VolumeLoadBalanceOnActualIOPS`：停用基於實際 IOPS 而非最小 IOPS 的切片均衡。

 a| 
細繩
 a| 
沒有任何
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。



== 請求範例

該方法的請求類似於以下範例：

[listing]
----
{
  "method": "DisableFeature",
    "params": {
        "feature" : "VolumeLoadBalanceOnActualIOPS"
    },
    "id": 1
}
----


== 回應範例

此方法傳回類似以下範例的回應：

[listing]
----
{
  "id": 1,
  "result": {}
}
----
