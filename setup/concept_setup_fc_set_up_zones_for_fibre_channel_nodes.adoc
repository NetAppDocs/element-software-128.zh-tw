= 為光纖通道節點設定區域
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您使用光纖通道節點和SolidFire儲存節點建立新叢集時，節點的全球連接埠名稱 (WWPN) 位址可在 Web UI 中找到。您可以使用 WWPN 位址對光纖通道交換器進行區域劃分。

當您建立包含節點的新叢集時，WWPN 會在系統中註冊。在 Element UI 中，您可以從「叢集」標籤存取的「FC 連接埠」標籤的「WWPN」欄位中找到 WWPN 位址。
