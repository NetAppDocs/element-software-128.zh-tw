= 容量詳情
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「管理」標籤上的「磁碟區」頁面提供有關活動磁碟區的信息，例如名稱、帳戶、與磁碟區關聯的存取群組以及磁碟區的大小。

* *ID*
+
系統產生的磁碟區 ID。

* *姓名*
+
該卷創建時所起的名稱。

* *帳戶*
+
指派給該磁碟區的帳戶名稱。

* *訪問群組*
+
卷所屬的磁碟區存取群組的名稱。

* *使用權*
+
建立磁碟區時指派給該磁碟區的存取類型。可能的值：

+
** 讀/寫：所有讀寫操作均被接受。
** 唯讀：允許所有讀取操作；不允許寫入操作。
** 已鎖定：僅允許管理員存取。
** ReplicationTarget：指定為複製磁碟區對中的目標磁碟區。


* *用過的*
+
體積中已使用空間的百分比。

* *尺寸*
+
卷的總大小（以 GB 為單位）。

* *快照*
+
為該磁碟區建立的快照數量。

* *服務品質策略*
+
使用者自訂 QoS 策略的名稱和連結。

* *最低IOPS*
+
該卷保證的最低 IOPS 數。

* *最大IOPS*
+
該卷允許的最大 IOPS 數。

* *突發IOPS*
+
在短時間內允許該卷的最大 IOPS 數。預設值 = 15,000。

* *屬性*
+
透過 API 方法以鍵/值對的形式指派給磁碟區的屬性。

* *512e*
+
指示磁碟區上是否啟用了 512e。可能的值：

+
** 是的
** 不


* 創建於*
+
卷冊的創建日期和時間。
