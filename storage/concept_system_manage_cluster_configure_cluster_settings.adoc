= 配置叢集設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從 Element UI 的「叢集」標籤查看和變更叢集範圍的設置，並執行叢集特定的任務。

您可以設定叢集滿度閾值、支援存取、靜態加密、虛擬磁碟區、 SnapMirror和 NTP 廣播用戶端等設定。
