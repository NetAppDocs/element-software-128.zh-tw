= 使用磁碟區快照進行資料保護
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
卷快照是磁碟區在特定時間點的副本。您可以對磁碟區進行快照，並在以後需要將磁碟區回滾到建立快照時的狀態時使用該快照。

快照類似於磁碟區克隆。但是，快照只是卷元資料的副本，因此您無法掛載或寫入它們。建立磁碟區快照也只需要少量系統資源和空間，因此快照建立速度比複製更快。

您可以對單一磁碟區或一組磁碟區進行快照。

（可選）將快照複製到遠端叢集，並將其用作磁碟區的備份副本。這樣，您就可以使用複製的快照將磁碟區回滾到特定的時間點。或者，您可以從複製的快照建立磁碟區的複製。
