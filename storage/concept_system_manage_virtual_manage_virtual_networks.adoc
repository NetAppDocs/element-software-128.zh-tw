= 管理虛擬網絡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SolidFire儲存中的虛擬網路允許將位於不同邏輯網路上的多個用戶端之間的流量連接到一個叢集。透過使用 VLAN 標記，叢集的連線在網路堆疊中被隔離。
