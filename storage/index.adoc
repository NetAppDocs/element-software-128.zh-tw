= 使用 Element 軟體管理存儲
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用 Element 軟體設定SolidFire存儲，監控叢集容量和效能，並管理跨多租戶基礎架構的儲存活動。

Element 是SolidFire叢集的核心儲存作業系統。  Element 軟體在叢集中的所有節點上獨立運行，使叢集中的節點能夠組合資源，並作為單一儲存系統呈現給外部用戶端。  Element 軟體負責整個系統的叢集協調、擴展和管理。
