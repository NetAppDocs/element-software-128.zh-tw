= 編輯 VRF VLAN
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更 VRF VLAN 屬性，例如 VLAN 名稱、子網路遮罩、閘道和 IP 位址區塊。

. 點選“集群”>“網路”。
. 按一下要編輯的 VLAN 的「操作」圖示。
. 按一下“*編輯*”。
. 在「編輯 VLAN」對話方塊中輸入 VRF VLAN 的新屬性。
. 點選「儲存變更」。
