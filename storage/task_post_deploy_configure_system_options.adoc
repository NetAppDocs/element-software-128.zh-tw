= 部署後配置SolidFire系統選項
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
設定好SolidFire系統後，您可能需要執行一些可選任務。

如果您變更了系統中的憑證，您可能需要了解這對其他元件的影響。
