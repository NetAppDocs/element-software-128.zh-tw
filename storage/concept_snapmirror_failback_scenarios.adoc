= SnapMirror故障恢復場景
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapMirror災難復原功能透過兩種故障復原場景進行說明。這些假設是原有的關係已經破裂（失敗）。

為便於參考，已列出對應流程中的步驟。


NOTE: 在此範例中，R1 = 原始關係，其中執行NetApp Element軟體的叢集是原始來源磁碟區 (Element)，而ONTAP是原始目標磁碟區 (ONTAP)。  R2 和 R3 代表透過反向重新同步操作所建立的反向關係。
