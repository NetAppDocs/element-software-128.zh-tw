= 編輯發起者
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以變更現有發起程式的別名，或在別名尚不存在時新增別名。

若要為每個發起方新增 CHAP 帳戶名稱和憑證，您必須使用 `ModifyInitiator`呼叫 API 來移除和新增 CHAP 存取權和屬性。

看link:../api/concept_element_api_about_the_api.html["使用 Element API 管理存儲"]。

.步驟
. 點選“管理”>“發起者”。
. 點選要編輯的發起者的「操作」圖示。
. 按一下“*編輯*”。
. 在「別名」欄位中輸​​入發起者的新別名。
. 點選「儲存變更」。
