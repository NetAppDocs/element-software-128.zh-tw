= 使用群組快照進行資料保護任務
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以建立一組相關磁碟區的群組快照，以保留每個磁碟區的元資料在特定時間點的副本。將來，您可以將群組快照用作備份或回滾，以將磁碟區組的狀態還原到先前的狀態。
