= 啟用虛擬路由和轉送
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以啟用虛擬路由和轉送 (VRF)，它允許路由器中存在多個路由表實例並同時工作。此功能僅適用於儲存網路。

只能在建立 VLAN 時啟用 VRF。如果要切換回非 VRF 模式，則必須刪除並重新建立 VLAN。

. 點選“集群”>“網路”。
. 若要在新 VLAN 上啟用 VRF，請選擇「建立 VLAN」。
+
.. 請輸入新VRF/VLAN的相關資訊。請參閱「新增虛擬網路」。
.. 選取“啟用VRF”複選框。
.. *可選*：輸入網關。


. 點選「建立VLAN」。
