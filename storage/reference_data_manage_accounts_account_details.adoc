= 帳戶詳情
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
管理標籤上的「帳戶」頁面提供了系統中每個帳戶的信息，例如 ID、使用者名稱以及分配給每個帳戶的磁碟區的效率詳細資訊。

* *ID*
+
系統產生的帳戶ID。

* *使用者名稱*
+
帳戶創建時所賦予的名稱。

* *地位*
+
帳戶狀態。可能的值：

+
** 活躍帳戶：一個活躍的帳戶。
** 已鎖定：一個已鎖定的帳戶。
** 已刪除：已刪除並清除的帳戶。


* *活躍卷數*
+
分配給該帳戶的活躍卷數。

* *壓縮*
+
分配給該帳戶的捲的壓縮效率得分。

* *去重*
+
分配給該帳戶的磁碟區的重複資料刪除效率分數。

* *精簡配置*
+
分配給該帳戶的捲的精簡配置效率得分。

* *整體效率*
+
分配給該帳戶的業務量的整體效率得分。
