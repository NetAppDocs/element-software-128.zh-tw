= 刪除虛擬網絡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除虛擬網路物件。在刪除虛擬網路之前，必須先將位址區塊新增至另一個虛擬網路。

. 點選“集群”>“網路”。
. 按一下要刪除的 VLAN 旁邊的「操作」圖示。
. 按一下“*刪除*”。
. 確認訊息。
