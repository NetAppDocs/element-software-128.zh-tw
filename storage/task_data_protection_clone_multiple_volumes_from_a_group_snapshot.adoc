= 從群組快照克隆多個磁碟區
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從某個時間點的群組快照克隆一組磁碟區。此操作要求磁碟區的群組快照已存在，因為群組快照將用作建立磁碟區的基礎。建立磁碟區之後，就可以像使用系統中的任何其他磁碟區一樣使用它們。

磁碟區大小和目前叢集負載會影響完成克隆操作所需的時間。

. 點選「資料保護」>「群組快照」。
. 按一下要用於磁碟區複製的群組快照的「操作」圖示。
. 在出現的選單中，選擇「從群組快照複製磁碟區」。
. 在「從群組快照複製磁碟區」對話方塊中輸入「新磁碟區名稱前綴」。
+
此前綴將應用於從群組快照建立的所有磁碟區。

. *可選：* 選擇一個不同的帳戶，克隆帳戶將屬於該帳戶。
+
如果您不選擇帳戶，系統會將新磁碟區指派給目前磁碟區帳戶。

. *可選：*為複製中的磁碟區選擇不同的存取方式。
+
如果您未選擇存取方式，系統將使用目前磁碟區存取方式。

. 點選“開始克隆”。
