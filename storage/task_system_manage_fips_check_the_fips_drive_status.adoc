= 檢查 FIPS 驅動器狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用NetApp Element軟體檢查叢集上是否啟用了 FIPS 磁碟機功能。 `GetFeatureStatus` API 方法，用於顯示 FIPS 磁碟機啟用狀態是真還是假。

. 使用 Element API，透過輸入以下命令檢查叢集上的 FIPS 磁碟機功能：
+
`GetFeatureStatus`

. 回顧結果 `GetFeatureStatus`API呼叫。如果 FIPS 磁碟機啟用值為 True，則啟用 FIPS 磁碟機功能。
+
[listing]
----
{"enabled": true,
"feature": "FipsDrives"
}
----
