= 快照日程詳情
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在「資料保護 > 計畫」頁面上，您可以在快照計畫清單中查看以下資訊。

* *ID*
+
系統產生的快照 ID。

* *類型*
+
日程安排類型。目前僅支援快照類型。

* *姓名*
+
此日程表創建時所起的名稱。快照計劃名稱最多可包含 223 個字符，並且包含 az、0-9 和破折號 (-) 字符。

* *頻率*
+
此日程安排的運行頻率。頻率可以按小時和分鐘、週或月設定。

* *再次發生的*
+
說明該日程安排是只運行一次還是定期運行。

* *手動暫停*
+
指示日程是否已手動暫停。

* 卷 ID
+
此調度程序運行時將使用的磁碟區的 ID。

* *最後一次運行*
+
上次運行該日程表的時間。

* *上次運轉狀態*
+
上次計劃執行的結果。可能的值：

+
** 成功
** 失敗
