= 停用「使用條款」橫幅
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以關閉「使用條款」橫幅。停用橫幅後，使用者在使用 Element UI 時不再需要接受使用條款。

* 您必須擁有叢集管理員權限才能設定使用條款。
* 請確保已啟用“使用條款”。
+
.. 點選「使用者」>「使用條款」。
.. 按一下“*禁用*”。
