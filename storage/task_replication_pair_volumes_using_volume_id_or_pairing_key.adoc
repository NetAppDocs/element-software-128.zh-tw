= 使用卷 ID 或配對密鑰配對卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
配對過程涉及使用磁碟區 ID 或配對金鑰將兩個磁碟區配對。

. 選擇以下方法之一來計算交易對數量：
+
** 使用磁碟區 ID：如果您擁有對要配對磁碟區的兩個叢集的叢集管理員存取權限，請使用此方法。此方法使用遠端叢集上磁碟區的磁碟區 ID 來發起連線。
** 使用配對金鑰：如果您只有來源叢集的叢集管理員權限，請使用此方法。此方法會產生一個配對密鑰，該密鑰可用於遠端叢集以完成磁碟區對。
+

NOTE: 磁碟區配對密鑰包含磁碟區資訊的加密版本，可能包含敏感資訊。請僅以安全的方式共用此金鑰。
