= 排查 FIPS 驅動器功能故障
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用NetApp Element軟體使用者介面，您可以查看與 FIPS 驅動器功能相關的系統叢集故障或錯誤訊息的警報。

. 使用 Element UI，選擇“報告”>“警報”。
. 尋找集群故障，包括：
+
** FIPS驅動程式不符
** FIPS 導致不合規


. 有關解決方法建議，請參閱群集故障碼資訊。
