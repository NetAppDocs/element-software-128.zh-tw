= 使用每個節點的儲存節點實用程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果NetApp Element軟體 UI 中的標準監控工具無法提供足夠的故障排除訊息，您可以使用每個節點的實用程式來排查網路問題。每個節點的實用程式提供特定的資訊和工具，可以幫助您排查節點之間或與管理節點之間的網路問題。
