= 啟用 FIPS 驅動器功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用NetApp Element軟體啟用 FIPS 驅動器功能。 `EnableFeature` API 方法。

叢集上必須啟用靜態資料加密，且所有節點和磁碟機都必須具備 FIPS 功能，當 GetFipsReport 對所有節點顯示「就緒」狀態時，即可表示這一點。

.步
. 使用 Element API，透過輸入以下命令在所有磁碟機上啟用 FIPS：
+
`EnableFeature params: FipsDrives`
