= 清除卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當一個磁碟區被清除時，它會從系統中永久刪除。該卷中的所有資料都遺失了。

系統會在刪除卷八小時後自動清除這些磁碟區。但是，如果您想在預定時間之前清除卷，您可以這樣做。


NOTE: 如果一次清除大量磁碟區，或要清除的每個磁碟區都有許多關聯的快照，則清除程序可能會失敗並傳回錯誤“xDBConnectionLoss”，該錯誤將顯示在 Element Web UI 的 API 日誌中。如果發生這種情況，請嘗試使用較少的容量再次執行清除操作。

. 選擇“管理”>“卷”。
. 點擊“已刪除”按鈕。
. 執行下列步驟以清除單一磁碟區或多個磁碟區。
+
|===


| 選項 | 步驟 


 a| 
清除單一磁碟區
 a| 
.. 按一下要清除的磁碟區的「操作」圖示。
.. 點選“清除”。
.. 確認操作。




 a| 
清除多個卷
 a| 
.. 選擇要清除的磁碟區。
.. 點擊“批量操作”。
.. 在出現的選單中，選擇“清除”。
.. 確認操作。


|===
