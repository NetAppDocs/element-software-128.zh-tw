= 刪除一個簇對
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從元素 UI 中刪除該對中的任一簇。

. 點選「資料保護」>「集群對」。
. 點選集群對的「操作」圖示。
. 在出現的選單中，按一下「刪除」。
. 確認操作。
. 從集群配對中的第二個集群再次執行這些步驟。
