= 驅動器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
儲存節點包含一個或多個實體驅動器，用於儲存叢集的一部分資料。將硬碟成功加入叢集後，叢集即可利用硬碟的容量和效能。

儲存節點包含兩種類型的磁碟機：

* 捲元資料驅動器
+
這些驅動器儲存壓縮訊息，用於定義叢集中的每個磁碟區、克隆或快照。系統中元資料驅動器的總容量決定了可以配置為磁碟區的最大儲存量。可配置的最大儲存量與叢集區塊磁碟機上實際儲存的資料量無關。卷元資料驅動器使用雙螺旋資料保護在叢集中冗餘儲存資料。

+

NOTE: 某些系統事件日誌和錯誤訊息將卷元資料磁碟機稱為切片磁碟機。

* *區塊驅動器*
+
這些驅動器儲存伺服器應用程式磁碟區的壓縮、去重資料塊。系統儲存容量的大部分由區塊驅動器構成。對SolidFire叢集上已儲存的資料的讀取請求以及寫入資料的請求大多發生在區塊磁碟機上。系統中的總塊驅動器容量決定了可以儲存的最大資料量，同時考慮了壓縮、精簡配置和去重的影響。
