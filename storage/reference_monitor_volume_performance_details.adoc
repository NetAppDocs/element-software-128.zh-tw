= 銷量表現詳情
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 Element UI 的「報告」標籤的「磁碟區效能」頁面中查看磁碟區的效能統計資料。

以下列表列出了您可以獲得的詳細資訊：

* *ID*
+
系統產生的磁碟區 ID。

* *姓名*
+
該卷創建時所起的名稱。

* *帳戶*
+
指派給該磁碟區的帳戶名稱。

* *訪問群組*
+
卷所屬的磁碟區存取群組的名稱。

* *容量利用率*
+
表示客戶使用容量大小的百分比值。

+
可能的值：

+
** 0 = 客戶端未使用卷
** 100 = 客戶端正在使用最大值
** >100 = 客戶端正在使用突發模式


* *總IOPS*
+
目前正在對磁碟區執行的 IOPS（讀取和寫入）總數。

* *讀取IOPS*
+
目前正在對該磁碟區執行的讀取 IOPS 總數。

* *寫入IOPS*
+
目前正在對該磁碟區執行的寫入 IOPS 總數。

* 總吞吐量
+
目前正在對磁碟區執行的總吞吐量（讀取和寫入）。

* *讀取吞吐量*
+
目前正在對該磁碟區執行的總讀取吞吐量。

* *寫入吞吐量*
+
目前正在對該磁碟區執行的總寫入吞吐量。

* 總延遲
+
完成對磁碟區的讀取和寫入操作的平均時間（以微秒為單位）。

* *讀取延遲*
+
在過去 500 毫秒內完成對磁碟區的讀取操作的平均時間（以微秒為單位）。

* *寫入延遲*
+
在過去 500 毫秒內完成對磁碟區的寫入操作的平均時間（以微秒為單位）。

* *隊列深度*
+
卷上未完成的讀取和寫入操作的數量。

* *平均 I/O 大小*
+
最近 500 毫秒內對磁碟區進行的最新 I/O 的平均大小（以位元組為單位）。
