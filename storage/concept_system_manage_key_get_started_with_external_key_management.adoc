= 開始使用外部密鑰管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
外部金鑰管理 (EKM) 結合叢集外外部金鑰伺服器 (EKS) 提供安全的驗證金鑰 (AK) 管理。  AK 用於在需要時鎖定和解鎖自加密磁碟機 (SED)。link:../concepts/concept_solidfire_concepts_security.html["靜態加密"]已在叢集上啟用。 EKS 提供 AK 的安全產生和儲存。此叢集利用金鑰管理互通協定 (KMIP)（OASIS 定義的標準協定）與 EKS 通訊。
